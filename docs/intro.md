# Intro

> `mlcast-datasets`: a collection of source datasets for machine-learning based nowcasting

You are currently looking at the Jupyter Book for the [`mlcast-datasets` intake catalog](). This catalog was created and is maintained by the MLCast Community [WG6 of the EUMETNET E-AI Optional Programme](https://github.com/eumetnet-e-ai/wg6_nowcasting).

The intention by behind this Jupyter Book is to:

1. Detail what a given dataset contains. This includes the variables defined, the source of data, the spatial and temporal resolution, and the time period covered.
2. Detail how to access a given dataset through the intake catalog, how to load it, and how to work with the data.
